
Text-based flash cards

Input file should have a top section that represents the front of flash cards, numbered

The top and bottom sections should be separated by a line with at least 5 hash marks (#)

The bottom section represents the back of flash cards, and should have numbers that correspond to the top section

e.g.
01. la tienda
02. rojo
#####
01. the store
02. red

The input file should be on the command line

Example
  ruby flash_cards.rb notes_spanish.txt
